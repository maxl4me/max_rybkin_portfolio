# Rybkin Maxim — QA Engineer · Portfolio

**Live site → [maxl4me.github.io/max_rybkin_portfolio](https://maxl4me.github.io/max_rybkin_portfolio/)**

My personal QA portfolio: background, skills, work experience, and featured
automation projects. Single-page, e-ink-inspired design (paper + ink, monospace).

## Featured project

**[DJI Storefront — UI Automation Framework](https://github.com/maxl4me/dji-automation)** —
a production-grade Playwright + Pytest framework (Page Object Model, data-driven
tests, Allure reporting, GitHub Actions CI) testing DJI's marketing site and online
store. Full test documentation (STD, STP, STR) lives in that repo.

## Stack

Hand-built **HTML + CSS** (no framework), Google Fonts (JetBrains Mono),
hosted on **GitHub Pages**. Vanilla JS for the CV-download control and
back-to-top button.

## Run locally

It's a static site — no build step:

```bash
git clone https://github.com/maxl4me/max_rybkin_portfolio.git
cd max_rybkin_portfolio
# open index.html in a browser, or serve it:
python -m http.server 8000   # then visit http://localhost:8000
```
